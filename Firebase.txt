◆ Googleの Firebaseデータベースの利用

Firebase Console: https://console.firebase.google.com/

1. Firebase Console にアクセスして、プロジェクトを作成
2. 左メニューの「Build」→「Realtime Database」を選択し、「データベースを作成」
	ロケーションは「米国（us-central1）」などでOK
	セキュリティルールはテスト用として「テストモード（30日間誰でも読み書き可能）」

3. プロジェクトの概要（歯車アイコン）→「プロジェクトの設定」→「マイアプリ」で「ウェブアプリ（</>）」を追加

4. 表示される firebaseConfig の内容（apiKeyや projectIdなどが含まれる部分）をコピーして、initializeApp()に指定

※ 「Firebase Hosting へのデプロイ」は行なう必要なし
Firebase Hosting は「Webページの置き場所」を提供するサービス
GitHub Pages を使わずに、ここだけで完結させるのであれば設定する

※ 30日経過後に使用できなくなるため期間の設定
1. Firebaseコンソールを開く
2. 左メニューの 「Build」→「Realtime Database」 を選択
3. 画面上部のタブから 「ルール (Rules)」 タブを選択
4. 期限なしに設定

{
  "rules": {
    ".read": "now < 1771858800000",  // 2026-2-24
    ".write": "now < 1771858800000",  // 2026-2-24
  }
}

↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

※ ただし、「世界中の誰でも、URLを知っていれば読み書きできる」状態。
使用しなくなったら falseに設定する
